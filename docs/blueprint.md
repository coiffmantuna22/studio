# **App Name**: SubFinder

## Core Features:

- Teacher Profile Management: Create and manage detailed teacher profiles, including subjects taught and individual calendars.
- Absence/Unavailability Tracking: Mark teachers as absent or temporarily unavailable, specifying dates and reasons.
- Substitute Teacher Recommendation: Suggest qualified substitute teachers based on availability, subject expertise, and teacher preferences. A reasoning tool will make the determination.
- Admin dashboard: Overview panel to help manage teacher absence data, profiles and subs.

## Style Guidelines:

- Primary color: Professional blue (#2962FF) to convey trust and reliability.
- Background color: Light blue (#E6F0FF) for a clean, calming interface.
- Accent color: Slightly darker blue (#3D5AFE) to highlight key actions and elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look; suitable for headlines or body text.
- Use simple, clear icons for teacher profiles, absence management, and recommendations.
- Clean and intuitive layout to ensure easy navigation and a user-friendly experience.
